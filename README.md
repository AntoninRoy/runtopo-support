# RunTopo - Support

Bienvenue sur la page d'assistance de RunTopo.

## Questions fréquentes

### Comment importer une trace depuis Strava ?
1. Ouvrez l'app et allez sur l'écran d'import
2. Appuyez sur "Importer depuis Strava"
3. Connectez-vous à votre compte Strava si ce n'est pas déjà fait
4. Sélectionnez l'itinéraire à importer

### Comment importer un fichier GPX ?
1. Ouvrez l'app et allez sur l'écran d'import
2. Appuyez sur "Importer un fichier GPX"
3. Sélectionnez le fichier depuis votre appareil

### Comment fonctionnent les conseils IA ?
RunTopo analyse automatiquement votre parcours et détecte les montées, descentes et portions plates. Pour chaque section, l'IA génère des conseils adaptés à la difficulté et aux caractéristiques du terrain.
