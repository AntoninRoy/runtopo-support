# Politique de confidentialité - RunTopo

*Dernière mise à jour : Mars 2025*

## Introduction

RunTopo ("nous", "notre", "l'application") respecte votre vie privée. Cette politique explique quelles données nous collectons et comment nous les utilisons.

## Données collectées

### Données de compte
- Identifiant utilisateur anonyme (généré automatiquement)
- Adresse email (uniquement si vous choisissez de vous connecter)

### Données d'utilisation
- Traces GPX importées (stockées sur nos serveurs pour l'analyse)
- Historique de vos parcours analysés

### Données Strava
Si vous connectez votre compte Strava, nous accédons à :
- Vos itinéraires Strava (lecture seule)
- Nous ne modifions jamais vos données Strava

## Utilisation des données

Nous utilisons vos données pour :
- Analyser vos parcours et générer des conseils personnalisés
- Sauvegarder votre historique de traces
- Améliorer nos algorithmes d'analyse

## Partage des données

Nous ne vendons pas vos données. Nous utilisons des services tiers :
- **Supabase** : hébergement et base de données
- **RevenueCat** : gestion des abonnements
- **Mixpanel** : analyse d'usage anonymisée
- **Sentry** : rapports de bugs (anonymisés)

## Conservation des données

Vos données sont conservées tant que votre compte est actif. Vous pouvez demander la suppression de vos données à tout moment.

## Vos droits

Vous pouvez :
- Accéder à vos données
- Demander la suppression de votre compte
- Exporter vos traces
